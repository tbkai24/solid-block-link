# Content Model

## Admin-managed fields

- Site hero title
- Site hero summary
- Primary CTA label and URL
- Secondary CTA label and URL
- Current campaign title
- Current campaign summary
- Donation goal
- Internal donation total
- Donor count
- Progress percentage
- Featured updates
- Featured embeds
- Past campaigns

## Data split

- Google Sheets + Apps Script: public donations and donor deduplication
- Admin panel: goals, internal donations, curated updates, post embeds, campaign records

## Homepage content order

1. Hero
2. Current campaign
3. Donation progress
4. Latest updates
5. Featured posts
6. Past campaigns
